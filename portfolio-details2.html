Magang Data Management di PT Telkom Indonesia

Pengalaman Magang di PT Telkom Indonesia

Selama 3 bulan (Oktober 2023 - Januari 2024) saya berkesempatan magang di PT Telkom Indonesia Witel Bekasi pada unit Access Data Management. Pengalaman ini memberikan saya pemahaman mendalam tentang manajemen data jaringan telekomunikasi dan sistem operasional perusahaan telekomunikasi terbesar di Indonesia.

Tugas dan Tanggung Jawab

  • Validasi dan migrasi data jaringan ke sistem INFRACO
  • Melakukan pointing dan kalkulasi pemasangan jaringan
  • Memastikan akurasi data infrastruktur jaringan di database
  • Berkolaborasi dengan tim teknis untuk update data jaringan
  • Membantu dalam pembuatan laporan analisis data jaringan

Pencapaian dan Kompetensi

Selama magang, saya berhasil memvalidasi lebih dari 500 data point jaringan dan berkontribusi dalam memperbaiki akurasi database sebesar 15%. Saya juga mendapatkan pemahaman praktis tentang:

  • Sistem manajemen data jaringan telekomunikasi
  • Prosedur operasional standar di perusahaan BUMN
  • Tools dan software khusus untuk manajemen data jaringan
  • Kerjasama tim dalam lingkungan kerja profesional

Selama masa magang, Raden Ahmad menunjukkan kemampuan analisis data yang baik dan ketelitian dalam pekerjaan. Kontribusinya dalam validasi data jaringan sangat membantu tim kami.
