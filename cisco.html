Sertifikasi Cisco Network Security

Cisco Networking Academy - Network Security

Sertifikasi ini membuktikan kemampuan saya dalam mengamankan infrastruktur jaringan menggunakan solusi Cisco. Saya menyelesaikan kursus intensif yang mencakup berbagai aspek keamanan jaringan modern melalui Cisco Networking Academy di Universitas Bina Sarana Informatika.

Kompetensi yang Dikuasai

Keamanan Router Cisco
Cisco IOS Firewall

IPsec VPN
Access Control Lists
Desain Jaringan Aman
Cisco ASA Firewall
Enkripsi Data

Detail Materi Pelatihan

  • Fundamental Keamanan Jaringan: Ancaman modern dan teknik mitigasi
  • Pengamanan Perangkat Cisco: Konfigurasi router dan switch yang aman
  • Firewall Implementation: Cisco IOS Firewall dan Zone-Based Policy Firewall
  • Network Authentication: Implementasi AAA menggunakan database lokal dan server eksternal
  • Virtual Private Networks: Membangun koneksi aman dengan IPsec VPN
  • Layer 2 Security: Mitigasi serangan pada lapisan data link
  • Cisco ASA: Deployment untuk firewall dan NAT/PAT services

Penerapan Praktis

Pengetahuan dari sertifikasi ini telah saya terapkan dalam berbagai proyek jaringan, termasuk:

  • Perancangan jaringan aman untuk lingkungan perkantoran
  • Implementasi firewall untuk melindungi sistem dari serangan eksternal
  • Konfigurasi VPN untuk koneksi aman antar kantor
  • Audit keamanan jaringan dan rekomendasi perbaikan
